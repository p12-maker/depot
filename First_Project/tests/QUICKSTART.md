# 🧪 系统监控程序 - 测试快速开始

## 🎯 5分钟快速上手

### 第1步：安装测试依赖（如果还没有）

```bash
pip install pytest pytest-cov
```

### 第2步：运行测试

**最简单的方式：**
```bash
run_tests.bat
```

然后选择 `[1]` 运行所有测试。

**或者使用命令行：**
```bash
pytest tests/ -v
```

### 第3步：查看结果

你会看到类似这样的输出：

## 💡 常见问题

### Q: 测试失败怎么办？
A: 查看错误信息，大多数情况下是环境问题。GUI 测试在没有 PyQt5 时会自动跳过。

### Q: 如何只运行一个测试？
```bash
pytest tests/test_system_monitor.py::TestSystemMonitor::test_get_cpu_info_success -v
```

### Q: 测试需要真实硬件吗？
A: 不需要！所有外部依赖都已 Mock，可以在任何环境运行。

### Q: 如何提高测试覆盖率？
```bash
# 生成覆盖率报告
pytest tests/ --cov=. --cov-report=html
# 打开 htmlcov/index.html 查看哪些代码未覆盖
```
