## 🔧 自定义和扩展

### 添加新测试
1. 在 `tests/` 目录创建 `test_<module>.py`
2. 编写测试类和方法
3. 在 `run_tests.py` 中添加模块名

### 运行特定测试
```bash
# 单个测试文件
pytest tests/test_system_monitor.py -v

# 单个测试类
pytest tests/test_system_monitor.py::TestSystemMonitor -v

# 单个测试方法
pytest tests/test_system_monitor.py::TestSystemMonitor::test_get_cpu_info_success -v
```

### 查看覆盖率
```bash
# 生成HTML报告
pytest --cov=. --cov-report=html

# 打开报告
start htmlcov\index.html
```

## ⚠️ 注意事项

### 环境要求
- Python 3.7+
- pytest（推荐）或 unittest
- PyQt5（可选，用于GUI测试）

### 自动跳过
- 无PyQt5时跳过GUI测试
- 不影响其他测试运行

### 权限安全
- 不会修改实际系统设置
- 所有操作都被Mock
- 文件系统操作用临时目录
